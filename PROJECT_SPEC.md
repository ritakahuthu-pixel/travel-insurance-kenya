# Travel Insurance Kenya

## Project Overview

Build a modern, production-ready travel insurance web application for Kenya.

The application should allow users to purchase affordable daily travel insurance using M-Pesa, manage their policies, and submit claims through an intuitive interface.

The design must be premium, clean, responsive, mobile-first, and comparable in quality to leading fintech products.
